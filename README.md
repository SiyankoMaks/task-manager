# 🔹 Task Manager

Веб-приложение для управления задачами с регистрацией и авторизацией пользователей.  
Позволяет создавать, просматривать и управлять задачами. Разработано для практики навыков веб-разработки на **Flask**, работы с базой данных **PostgreSQL**, шаблонами **Jinja2** и фронтендом на **Tailwind CSS** с **Alpine.js**.

---

## 🎯 Цель проекта
- Практика работы с Flask и Blueprints
- Освоение ORM SQLAlchemy и взаимодействия с PostgreSQL
- Создание адаптивного фронтенда с Tailwind CSS
- Реализация функционала регистрации, логина и управления задачами
- Интеграция flash-сообщений и анимаций через Alpine.js и CSS

---

## 🛠️ Используемые технологии
- **Python 3.x**
- **Flask**
- **Flask-Login**

## ⚙️ Установка и настройка

### 1. Клонировать репозиторий
```bash
git clone https://github.com/username/task-manager.git
cd task-manager
```

### 2. Создать виртуальное окружение
```bash
python -m venv venv
```

### 3. Активировать виртуальное окружение

**Windows**
```bash
venv\Scripts\activate
```

**Linux / macOS**
```bash
source venv/bin/activate
```

### 4. Установить зависимости
```bash
pip install -r requirements.txt
```

### 5. Инициализация базы данных
```python
python
>>> from app import app
>>> from extensions import db
>>> from models import User, Task
>>> with app.app_context():
...     db.create_all()
...     exit()
```

### 6. Запуск сервера
```bash
python app.py
```

Откройте браузер по адресу: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📚 Пример использования

1. **Регистрация пользователя**
```
Перейти на /register
Заполнить username, email и password
Нажать "Register"
```

2. **Вход на сайт**
```
Перейти на /login
Ввести username и password
Нажать "Login"
После успешного входа — перенаправление на /tasks
```

3. **Работа с задачами**
```
На странице /tasks:
- Добавление новой задачи через форму
- Просмотр списка задач пользователя
- Редактирование задач через /edit_task/<id>
```
