# 🔹 Task Manager

Веб-приложение для управления задачами с регистрацией и авторизацией пользователей.  
Позволяет создавать, редактировать, удалять и отслеживать выполнение задач.

Проект разработан для практики backend-разработки на **Flask**, работы с базой данных (**PostgreSQL / SQLite**), ORM (**SQLAlchemy**) и современного фронтенда (**Tailwind CSS**).

---

## 🎯 Цель проекта

- Практика разработки backend-приложений на Flask
- Работа с архитектурой (Blueprints, разделение слоёв)
- Освоение ORM SQLAlchemy и работы с БД
- Реализация полноценного CRUD (Create, Read, Update, Delete)
- Работа с авторизацией пользователей (Flask-Login)
- Построение удобного UI с Tailwind CSS

---

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/username/task-manager.git
cd task-manager
````

### 2. Создать виртуальное окружение

```bash
python -m venv venv
```

### 3. Активировать окружение

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 4. Установить зависимости

```bash
pip install -r requirements.txt
```

### 5. Создать базу данных

```bash
python init_db.py
```

> ⚠️ Используется `drop_all()` — только для разработки

---

### 6. Запуск сервера

```bash
python app.py
```

Открыть: [http://127.0.0.1:5000](http://127.0.0.1:5000)

---

### ✅ Задачи (CRUD)

Пользователь может:

* ➕ Создавать задачи
* 📋 Просматривать список задач
* ✏️ Редактировать задачи
* ❌ Удалять задачи
* ✔️ Отмечать задачи как выполненные

---

### 🧩 Структура задачи

Каждая задача содержит:

* `title` — название
* `description` — описание
* `status` — статус (`pending / done`)
* `priority` — приоритет (`low / medium / high`)
* `deadline` — дедлайн
* `created_at` — дата создания
* `user_id` — владелец задачи

---

### 🔐 Безопасность

* Доступ к задачам только для авторизованных пользователей
* Защита данных через фильтрацию по `user_id`
* Невозможность получить доступ к чужим задачам
